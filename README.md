# 김휘웅 | SW QA 포트폴리오

## Mini Commerce QA Lab

커머스 서비스의 상품 조회, 품절 상품 주문 제한, 주문 수량 경계값, 장바구니/주문 흐름을 중심으로  
UI 동작, API 응답, DB 반영 결과를 함께 검증한 QA 포트폴리오입니다.

## 주요 검증 범위

- 상품 조회 및 상세 화면 검증
- 품절 상품 주문 제한 검증
- 주문 수량 경계값 검증
- Postman API 응답 확인
- H2 Console DB 데이터 정합성 확인
- 버그 리포트 작성
