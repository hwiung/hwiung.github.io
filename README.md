# 김휘웅 | SW QA 포트폴리오

## Mini Commerce QA Lab

커머스 서비스의 상품, 장바구니, 주문, 관리자 권한 흐름을 대상으로  
UI 동작, API 응답, DB 반영 결과, DevTools 저장소 값을 함께 확인한 QA 포트폴리오입니다.

## 핵심 검증 내용

- 상품 상세 및 장바구니 흐름 검증
- 바로 주문 생성 후 H2 DB 정합성 확인
- 품절 상품 주문 / 장바구니 처리 검증
- 수량 0, 음수, 재고 초과 경계값 테스트
- localStorage role 조작을 통한 관리자 화면 노출 이슈 확인
- USER token의 관리자 API 접근 403 응답 확인
